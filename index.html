A Vanilla JS Content Blocker

The JS feature takes in a string, and will check for swear/profane words based off of a provided list.

Four Languages are currently supported: English, Spanish, Italian, and Indonesian.

If you have a list of swear/profane words, submit a GitHub issue to GitHub.



Note: There is a slight delay in the demo. The package takes less than a second to parse the JSON files.

In a normal use case, the ContentBlocker will be imported when the page loads and language is set then, so timing will not be an issue.
